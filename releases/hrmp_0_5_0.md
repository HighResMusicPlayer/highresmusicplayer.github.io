# hrmp 0.5.0

This is a feature release.

## Features

* Digital Stream Digital (DSD) 64/128/256/512 (Sony .dsf)
* Digital Stream Digital (DSD) over Pulse Code Modulation (PCM) (DoP) 64/128/256 (Sony .dsf)
